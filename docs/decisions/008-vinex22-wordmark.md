# ADR-008 — Subtle `vinex22` wordmark on product/category shots only

**Date**: 2026-04-24
**Status**: Accepted

## Context
User requested in-image branding for product authenticity.

## Decision
Add a discreet lowercase `vinex22` wordmark (≤5% of frame) on product + category bucket images. Hero, lifestyle, texture, campaign images stay logo-free for editorial elegance. Implemented via `branded_buckets` in `image-gen/catalog.json`.

## Consequences
Stronger brand identity on PDP / catalog cards; clean editorial look on storytelling surfaces.
