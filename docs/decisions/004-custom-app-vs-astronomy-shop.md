# ADR-004 — Build a custom app instead of using Astronomy Shop

**Date**: 2026-04-24
**Status**: Accepted

## Context
Customer wants a differentiated demo. OTel Astronomy Shop is well-known and over-used in Microsoft demos.

## Decision
Build an Apple-inspired storefront — see [ADR-007](007-app-name-vinex22-travels.md) for the brand.

## Consequences
More control over branding and the planted code bug; more maintenance burden vs prebuilt.
