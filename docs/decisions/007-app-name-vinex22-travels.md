# ADR-007 — App name and theme: vinex22-travels (travel essentials)

**Date**: 2026-04-24
**Status**: Accepted

## Context
Need a memorable, on-brand demo app. Travel essentials (luggage, pillows, packing cubes) suit Apple-style minimalist photography and have natural product variety.

## Decision
Name = `vinex22-travels`. Categories: Carry · Rest · Pack · Care · Tech.

## Consequences
Clear theme for image generation prompts; rich catalog without needing huge SKU count.
