# ADR-002 — AKS Automatic SKU

**Date**: 2026-04-24
**Status**: Superseded by [ADR-011](011-aks-standard-sku.md)

## Context
Need a production-feel cluster with minimal config.

## Decision
AKS Automatic (Workload Identity, CNI Overlay, managed add-ons by default).

## Consequences
Less knob-twiddling; aligned with Microsoft-recommended path.
